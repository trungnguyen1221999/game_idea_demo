# 🗝️ The Golden Key Adventure - QR Code Game Demo
Link : [qrgameidea.netlify.app](https://qrgameidea.netlify.app/)
## 📖 Overview

The Golden Key Adventure is an interactive QR code-based treasure hunt game designed for educational environments, specifically demonstrated for Vuoksi school. Players follow Capi the Capybara on an enchanting journey to collect six golden key fragments by completing challenges at various checkpoints.

## 🎮 Game Description

### Story
Join Capi the Capybara on a magical adventure to rescue Princess Lily! Navigate through an enchanted forest, complete challenges at six mystical checkpoints, and collect golden key fragments. Each checkpoint presents unique puzzles and rewards players with coins that can be spent in the magical shop.

### Main Characters
- **🐹 Capi** - The brave Capybara hero
- **👸 Princess Lily** - The captured princess awaiting rescue
- **🐒 Moki** - The mischievous monkey guardian
- **🐱 Cleo** - The wise cat guardian

## 🚀 Getting Started

### Prerequisites
- Modern web browser
- QR code scanner (can use phone camera)
- Display device (tablet, smartphone, or computer)

### Installation
1. Download or clone the project files
2. Open `golden_key_game.html` in a web browser
3. No additional setup required - runs entirely in the browser

### How to Play
1. **Start the Game**: Open the HTML file in your browser
2. **Scan QR Codes**: Click on checkpoint circles or scan actual QR codes
3. **Complete Challenges**: Solve puzzles within the time limit
4. **Collect Rewards**: Earn coins and key fragments
5. **Use the Shop**: Purchase power-ups to enhance gameplay
6. **Rescue the Princess**: Complete all six checkpoints to win
